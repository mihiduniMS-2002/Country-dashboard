 **Global Country Insights Dashboard**

A full-stack web application that provides real-time insights about any country by integrating multiple public APIs.  
The system aggregates data for weather, country information, currency exchange rates, and air quality, allowing users to explore global information in one place.

This project was developed for the Service Oriented Computing (SOC) module.

---

@ Features:-

- Search any country and view complete live data  
- Weather & 5-day forecast (OpenWeatherMap)  
- Country details: capital, region, population, map, flag  
- Live currency exchange rates  
- Real-time air quality (OpenAQ API)  
- Optimized backend with caching (NodeCache)  
- API key protection via `.env`  
- Fully responsive UI  




@Tech Stack:-

(1).Frontend
- HTML5  
- CSS3  
- JavaScript (ES6)

(2).Backend
- Node.js  
- Express.js  
- node-fetch  
- dotenv  
- node-cache  
- CORS

(3)-APIs Used
- OpenWeatherMap API – Weather  
- RestCountries API – Country info  
- ExchangeRate API – Currency  
- OpenAQ API – Air Quality
